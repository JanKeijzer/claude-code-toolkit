# Shared Procedures

Reusable procedure snippets that can be included in any project's CLAUDE.md.

These are created via `/promote` when a project-local procedure proves useful across multiple projects. Each file is a self-contained procedure in markdown format.

To use in a project, either:
- Copy the relevant procedure into the project's CLAUDE.md
- Or reference it: "See ~/.claude/skills/claude-code-toolkit/claude-md/procedures/<name>.md"
